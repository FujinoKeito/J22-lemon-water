特面対

特面対とは

就活生が面接をしている際に面接官から変わった質問をされた場合に、

「特面対」

が参考になる解答を教えてくれて、
その解答を元に面接対策ができる特化型面接対策アプリです。
どの診断も10問だけで診断します。

使用方法

  1. 診断したい診断を選ぶ
  2. 3つの診断開始ボタンから1つを選びクリック
  3. 解答画面では、チェックボックスから1つを選択し、選択が完了すれば次へをクリック

    4. ∗戻りたい場合は、戻るボタンをクリック

  4. 質問を全て解答するとあなたの性格、動物、カラーなどの診断結果を「特面対」が教えてくれます。
